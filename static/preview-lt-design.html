Installing the OpenShift CLI

You can install the OpenShift CLI (oc) to interact with OpenShift Container Platform from a command-line interface. The following procedure describes how to install the CLI on Linux.

.Prerequisites

You have a running OpenShift Container Platform cluster and you have logged into it using a web console or API token. Ensure that you have sufficient privileges to perform the installation.

.Procedure

1. Navigate to the OpenShift Container Platform downloads page for your operating system and download the appropriate oc binary.

2. Unpack the archive by running the following command:

$ tar xvf oc.tar.gz

3. Place the oc binary in a directory that is on your PATH. You can verify this by running echo $PATH in your termnial.

4. After the installation is complete, verify by running:

$ oc version

This will display the version of the CLI and the server. If you cannot run this command, make sure your PATH variable includes, the directory where you placed the binary.

For more information about configuing the CLI, refer to the below documentation section.
