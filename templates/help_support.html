Provide Feedback

When reviewing content, use the thumbs-up/thumbs-down buttons on each issue card to tell us if a suggestion was helpful. This data improves rule accuracy over time.

Report Issues

Found a bug or have a feature request? Open an issue in our GitLab repository.

UI bugs: Include the browser console output and a screenshot.
False positives: Include the flagged text and the rule type.

Contact the Team

Reach out via Slack or email for questions, feature discussions, or integration support.

About CEA

CEA checks your technical writing against 107 deterministic rules based on the IBM Style Guide. All style checking is rules-based (no LLM). The optional AI rewriting feature uses a local model for privacy.

Supported formats: AsciiDoc, Markdown, DITA/XML, PDF, DOCX, plain text
